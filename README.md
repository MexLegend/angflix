## Cloning the repository

```shell
git clone https://github.com/MexLegend/angflix.git
```

## Getting Started

Install the dependencies:

```sh
$ pnpm i
# or
$ npm i
```

Start the app

```sh
$ ng s
# or
$ pnpm start
# or
$ npm run start
```

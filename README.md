## Cloning the repository

```shell
git clone https://github.com/MexLegend/angflix.git
```

## Getting Started

Install the dependencies:

```sh
$ pnpm i
```
or
```sh
$ npm i
```

Start the app

```sh
$ ng s
# or
$ pnpm start
# or
$ npm run start
```

## Running end-to-end tests

Run the apllication in host mode:

```sh
$ pnpm start:host
# or
$ npm run start:host
```

Run cypress test:

```sh
$ pnpm test:cypress
# or
$ npm test:cypress
```
